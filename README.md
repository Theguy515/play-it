Play it!

A 2D top-down arcade game collection Unity project built by Jordan Andrews and Gavin Keller for Software development 2. 

---

Features
- Custom tilemaps/art
- Lobby scene setup
- Arcade-style sprites
- Unity URP 2D rendering

---

Tech Stack
- Engine: Unity
- Language: C#
- Version Control: GitHub
- Pixelorama
- 
---

Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Play-it.git
   ```

2. In the Unity Hub, use the option "Add project from disk"
3. Open project
